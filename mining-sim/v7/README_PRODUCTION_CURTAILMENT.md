# Production Curtailment

This V7 layer reduces configured mining production loads before emergency UFLS.

It supports:
- continuously curtailable and discrete minimum-load assets;
- explicit production priority and safety-critical classification;
- reserve-gated staged restoration;
- separate production-loss accounting;
- event and sample evidence for every action.

It does not replace project-specific process studies, OEM limits, UFLS settings or protection coordination.
