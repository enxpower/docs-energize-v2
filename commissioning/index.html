How It Works

V1 Pipeline

Upload files
  ↓ parse + extract text
  ↓ chunk + tag metadata
  ↓ index (persistent, this browser)
Ask question
  ↓ retrieve relevant excerpts
  ↓ AI analysis with citations
Generate report

Documents are processed and stored locally in this browser (IndexedDB). Nothing is uploaded to any server. With an API key configured, document excerpts are sent directly to the Anthropic API for analysis.

Settings

Your API key is stored only in this browser (localStorage) and sent only to api.anthropic.com over HTTPS. It is never embedded in this site's code or sent to any other server. Without a key, the app runs in Mock mode — parsing, indexing, retrieval, and templates remain fully functional.
