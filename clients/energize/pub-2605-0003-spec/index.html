文档信息

1. 本文件目的

本文件专门用于指导 EMS 控制柜厂家修改柜门标签、端子表、线号表和图纸 wording。

目标：现场工程师看到柜门标签后，可以直接正确接线；不会误认为 EMS 需要直接接 PCS / BMS / STS / Fire；不会误认为 EMS DO1 是防孤岛保护跳闸；不会误认为 RS485 通道需要接 PCS / BMS / PV。

2. 项目上下文

本项目采用 All-in-one BESS 架构。
